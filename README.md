# 📚 Sistema de Biblioteca Web

## 🎯 Descripción
Sistema web para la administración de una biblioteca física que permite gestionar libros, autores, usuarios y préstamos con control de acceso por roles.

---

## 🛠️ Tecnologías
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Apache (XAMPP)

---

## 🔐 Funcionalidades
- Login con roles (administrador / usuario)
- CRUD de libros
- CRUD de autores
- CRUD de usuarios
- Gestión de préstamos y devoluciones
- Búsqueda y filtros de libros
- Control de stock de ejemplares
- Portadas de libros

---

## 🗄️ Base de datos
Incluye archivo: database/biblioteca



---

## ⚙️ Instalación
1. Clonar repositorio
2. Importar base de datos en phpMyAdmin
3. Colocar proyecto en `htdocs`
4. Ejecutar: http://localhost/biblioteca
